## Overview

Returns the integer smaller than or equal to `x`.

## Syntax

	math.floor( x )

##### x ~^(required)^~
_[Number][api.type.Number]._ A number.


## Examples

``````luq
print( math.floor(0.5) )    ---> 0
print( math.floor(-0.5) )    ---> -1
``````
