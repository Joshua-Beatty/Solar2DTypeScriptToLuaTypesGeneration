## Overview

Returns the hyperbolic tangent of `x`.

## Syntax

	math.tanh( x )

##### x ~^(required)^~
_[Number][api.type.Number]._ A number.


## Examples

``````lua
print( math.tanh( math.log(2) ) )   ---> 0.6
``````
