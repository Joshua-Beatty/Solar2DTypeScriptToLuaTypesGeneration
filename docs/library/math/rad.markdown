## Overview

Converts to radians an angle given in degrees.

## Syntax

	math.rad( x )

##### x ~^(required)^~
_[Number][api.type.Number]._ An angle in degrees.


## Examples

``````lua
print(math.rad(180))  ---->  3.1415926535898
print(math.rad(1))    ---->  0.017453292519943
``````
