# math.sqrt()

> --------------------- ------------------------------------------------------------------------------------------

> __Revision__          [REVISION_LABEL](REVISION_URL)

## Overview

Returns the square root of `x` (equivalent to the expression `x^0.5`).

## Syntax

	math.sqrt( x )

##### x ~^(required)^~
_[Number][api.type.Number]._ A number.


## Examples

``````lua
print (math.sqrt (100))    ---> 10
print (math.sqrt (-1))     ---> NaN
``````
