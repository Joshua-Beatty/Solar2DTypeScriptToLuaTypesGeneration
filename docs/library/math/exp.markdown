## Overview

Returns the value `e^x^`.

## Syntax

	math.exp( x )

##### x ~^(required)^~
_[Number][api.type.Number]._ The exponent.


## Examples

``````lua
print(math.exp(0))   ---> 1
print(math.exp(1))  ---> 2.718281828459
``````
