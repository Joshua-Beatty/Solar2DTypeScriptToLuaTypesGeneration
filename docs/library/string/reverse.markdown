## Overview

Reverses a string.

## Syntax

	string.reverse( s )

	s:reverse()

##### s ~^(required)^~
_[String][api.type.String]._ The string to reverse.


## Example

`````lua
print( string.reverse( "Corona" ) )  --> anoroC
`````
